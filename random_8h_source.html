31 /***************************************************
32 
33  Uniform RNGs
34 
35  These all return well-distributed unsigned 32-bit ints that
36  have been tested rigorously on the DIEHARD, dieharder and TestU01
37  test suites. The Marsaglia KISS generators are slightly faster
38  and the 64-bit one uses the theoretically superior multiplicative
39  algorithm. The WELL1024a has a cycle length of (2^1024)-1
40 
41  Two versions are available. A simple version which
42  has a global hidden seed which does not need to be set, and a custom
43  version which allows seeds to be set by the user to either
44 
45  1 - promote independence or
46  2 - guarantee identical behaviour
47 
48  ..across multiple random number streams.
49 
50  These generators can all be used with the non-uniform RNGs below
51 
52 ****************************************************/
